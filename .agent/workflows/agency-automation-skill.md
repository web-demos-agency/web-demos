# 🤖 AI SITE AGENCY AUTOMATION SKILL

Этот навык расширяет мои возможности по автоматизации маркетинга и коммуникаций для нашего агентства.

## 📡 1. Поиск лидов (Lead Gen Strategy)
При поиске новых клиентов (через инструмент `browser`) я теперь всегда ищу:
- Рейтинг 4.5+ (значит, бизнес живой и востребованный).
- Отсутствие сайта или старый сайт (неадаптивный).
- Имена владельцев/мастеров в отзывах (для персонализации).

## ☎️ 2. Сценарии для ИИ-звонилок (Bland.ai / Retell)
Для звонков я готовлю скрипты по структуре:
1. **Hook:** "Hi [Name], I noticed your 5-star ranking on Google Maps..."
2. **The Gift:** "I've already built a premium demo for your business to show you how much more you could be making."
3. **The Ask:** "Can I text you the link to check it out?"
4. **Closing:** "My name is [Designer Name] from AI Site Agency. Talk soon!"

## 💬 3. Формат SMS-рассылки
Короткий шаблон для быстрой отправки:
> "Hi [Name], it's Designer from AI Site Agency. Your 4.8 rating is amazing! I built a live demo for [Business Name] to help you get more premium leads: [Link]. Check it out!"

## 🛠 4. Инструментарий
- **SMS:** Использовать TextNow (через VPN) или Twilio API.
- **Calls:** Использовать Bland.ai API (создание агента с голосом "Professional Male/Female").
- **CRM:** Вести учет в `leads.md` (Статус: Проверено / Позвонили / Отправили SMS / Закрыто).
